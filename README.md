## ✨ Key Features

- **🌓 Dynamic Dark/Light Mode**: Seamlessly toggle between dark and light themes with persistent user preference storage.
- **🎭 GSAP-Powered Animations**: High-performance scroll-triggered reveals, entrance staggers, and interactive timeline animations.
- **📱 Ultra-Responsive Design**: Mobile-first architecture with a refined navigation drawer and adaptive layouts for all screen sizes.
- **🕒 Interactive Experience Timeline**: A vertically-synced, GSAP-animated career history tracking professional growth.
- **🎥 Project Showcase**: Detailed project cards with technology chips, GitHub integration, and video demo modals.
- **🎨 Modern Glassmorphism UI**: Premium visual language using backdrop blurs, subtle gradients, and custom shadows built with Tailwind CSS.
- **📝 Automated Typewriter Effect**: Engaging hero section intro using `Typed.js` for professional role display.

---

## 🛠️ Tech Stack

### Frontend & Design
- **Structure**: HTML5 (Semantic)
- **Styling**: Tailwind CSS (via CDN) & Vanilla CSS
- **Interactions**: Javascript (ES6+)
- **Animations**: GSAP (GreenSock), ScrollTrigger
- **Typography**: Typed.js (Role intro)
- **Icons**: Font Awesome 6.5.1

### Core Competencies (Showcased)
- **Backend**: Java, Spring Boot, Microservices, RESTful APIs, Node.js
- **Database**: MySQL, PostgreSQL, MongoDB
- **Cloud/DevOps**: AWS (EC2, S3, IAM), Docker, Kubernetes, Jenkins
- **Testing**: JUnit (Mocking, Coverage)

## 🚀 Getting Started

### Prerequisites
To run this project locally, you only need a modern web browser.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Arbaz4Sayyad/MyPortfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd PortFolio
   ```
3. Open `index.html` in your browser:
   - **Using VS Code Live Server (Recommended)**: Right-click `index.html` and select "Open with Live Server".
   - **Directly**: Simply double-click `index.html` in your file explorer.
